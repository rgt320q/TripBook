# Trip Book - README & User Guide (English)

## Overview
Trip Book is a mobile application designed to help you plan your travels, save locations, create routes, and track your progress during your journey.

## Features
*   User login and secure authentication (Firebase Auth).
*   Pinning and saving locations on the map.
*   Grouping locations (e.g., "Paris Trip", "Summer Vacation").
*   Creating and optimizing routes between selected locations.
*   Automatically fetching Wikipedia information for route stops.
*   Adding specific needs lists and notes for each location.
*   Viewing a combined needs list and separate private notes per route.
*   Saving and viewing past routes.

## Installation
To run the Trip Book application locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your_username/TripBook.git # Replace with your repo URL
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd TripBook
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the application:**
    ```bash
    flutter run
    ```
    *   For Firebase configuration, you might need to add `google-services.json` (Android) and `GoogleService-Info.plist` (iOS) files to your project.
    *   Don't forget to add your Google Maps API key to `AndroidManifest.xml` (Android) and `AppDelegate.swift` (iOS) files.

## Technologies Used
*   **Flutter:** Mobile application development framework.
*   **Dart:** Programming language used by Flutter.
*   **Firebase Authentication:** For user authentication processes.
*   **Cloud Firestore:** Real-time database service.
*   **Google Maps API:** For map display and location services.
*   **Geolocator:** For location services.
*   **Flutter Local Notifications:** For local notifications.
*   **Wikipedia API (assumed):** To fetch Wikipedia information for route stops.
*   **SQFlite:** For local database operations.
*   **URL Launcher:** To open external URLs.

## User Guide (English)

### 1. Overview
Trip Book is a mobile application designed to help you plan your travels, save locations, create routes, and track your progress during your journey.

---

### 2. Map Screen and Basic Operations
The main screen of the application is an interactive map.

#### 2.1. Location Markers (Pins)
- **Your Current Location:** Shown on the map as a blue dot with a glowing effect around it.
- **Saved Locations:** Displayed as classic Google Maps pins.
  - **Coloring:**
    - Locations assigned to a group take on the group's color.
    - Locations not belonging to any group are shown in **gray**.
    - Locations you have visited during an active route turn **green**.

#### 2.2. Adding a New Location
- You can add a new location by **long-pressing** anywhere on the map.
- In the pop-up window, you can enter details specific to the location, such as a name, description, notes, a list of needs, and estimated stop duration.

#### 2.3. Go to My Location Button
- By pressing the **target (my_location)** icon in the bottom right corner of the map, you can instantly center the map on your current location.

---

### 3. Location and Group Management
The application provides dedicated screens for efficiently managing your locations and groups.

#### 3.1. Manage Locations
- You can access this screen via the **list (list_alt)** icon in the top menu.
- All your saved locations are listed here.
- You can sort locations by name or creation date.
- Clicking on any location allows you to edit its details (name, description, group, notes, etc.).
- **Transition from Map:** When you click on a pin on the map, this screen opens, and the list automatically scrolls to the relevant location.

#### 3.2. Manage Groups
- You can access this screen via the **folder (folder_copy_outlined)** icon in the top menu.
- You can create, edit, and delete colored groups to categorize your locations.
- When editing a location, you can assign it to one of the groups you have created.

---

### 4. Route Planning and Usage
The application allows you to create dynamic routes from your saved locations.

#### 4.1. Creating a Route
- Click the **directions (directions)** icon in the top menu on the map screen.
- **Select from Group:** Creates a route including all locations in a specific group.
- **Manual Selection:** Allows you to create a custom route by selecting desired locations from a list.
- Once a route is created, the application sorts the locations most efficiently based on your current location.

#### 4.2. Saving and Loading Routes
- **Saving:** You can save the current route by naming it and pressing the **save (save)** icon on the route summary screen.
- **Loading:** By pressing the **route (route_sharp)** icon in the top menu on the map screen, you can load previously saved routes.

#### 4.3. Route Summary and Checklist
- When a route is created or for an active route, pressing the **summary (summarize)** icon in the top menu opens a summary screen.
- This screen displays information such as total distance, estimated travel, and stop durations.
- **Needs List:**
  - All needs added to locations on the route are presented here as a combined checklist.
  - This list is **specific to the current journey**. Items you check (ticks) are preserved until you close the route.
  - When the route is closed and reopened, or a new journey is started with the same route template, this list comes cleared (unticked).
- **Private Notes:** All private notes belonging to locations on the route are also displayed together on this screen.

---

### 5. Active Route Tracking
When you start a route, the map screen actively tracks your journey.

- **Path to Go:** Unvisited parts of your path are shown in **light gray** on the map.
- **Path Traveled:** When you reach a stop, the color of the path you have traveled up to that stop permanently turns **vibrant blue**.
- **Reached Stops:** When you get within 50 meters of a stop, that stop's pin permanently turns **green**.
- **Notifications:** When you approach a stop, you will receive a notification containing a Wikipedia summary (if available) related to that location. You will also receive a time reminder notification if you stay longer than planned at a stop.

#### 5.1. Reached Locations History
- You can access a list of all locations you have reached and received notifications for during a route, along with their Wikipedia links, via the **history (history)** icon in the top menu.

## Kullanıcı Kılavuzu (Türkçe)

### 1. Genel Bakış
Trip Book, seyahatlerinizi planlamak, konumları kaydetmek, rotalar oluşturmak ve seyahat esnasında ilerlemenizi takip etmek için tasarlanmış bir mobil uygulamadır.

---

### 2. Harita Ekranı ve Temel İşlemler
Uygulamanın ana ekranı interaktif bir haritadır.

#### 2.1. Konum İşaretçileri (Pinler)
- **Mevcut Konumunuz:** Haritada, etrafında parlama efekti olan mavi bir nokta ile gösterilir.
- **Kaydedilmiş Konumlar:** Klasik Google Haritalar pini şeklinde gösterilir.
  - **Renklendirme:**
    - Bir gruba atanmış konumlar, grubun rengini alır.
    - Herhangi bir gruba dahil olmayan konumlar **gri** renkte gösterilir.
    - Aktif bir rota sırasında ziyaret ettiğiniz konumlar ise **yeşil** renge döner.

#### 2.2. Yeni Konum Ekleme
- Haritada herhangi bir yere **uzun basarak** yeni bir konum ekleyebilirsiniz.
- Açılan pencerede konuma özel bir isim, açıklama, notlar, ihtiyaç listesi ve tahmini durma süresi gibi detayları girebilirsiniz.

#### 2.3. Konumuma Git Butonu
- Haritanın sağ alt köşesinde bulunan **hedef (my_location)** ikonuna basarak haritayı anında mevcut konumunuza ortalayabilirsiniz.

---

### 3. Konum ve Grup Yönetimi
Uygulama, konumlarınızı ve gruplarınızı verimli bir şekilde yönetmeniz için özel ekranlar sunar.

#### 3.1. Konumları Yönet
- Üst menüdeki **liste (list_alt)** ikonu ile bu ekrana ulaşabilirsiniz.
- Tüm kayıtlı konumlarınız burada listelenir.
- Konumları isme veya oluşturulma tarihine göre sıralayabilirsiniz.
- Herhangi bir konuma tıkladığınızda detaylarını (isim, açıklama, grup, notlar vb.) düzenleyebilirsiniz.
- **Haritadan Geçiş:** Haritadaki bir pine tıkladığınızda, bu ekran açılır ve liste otomatik olarak ilgili konuma kaydırılır.

#### 3.2. Grupları Yönet
- Üst menüdeki **klasör (folder_copy_outlined)** ikonu ile bu ekrana ulaşabilirsiniz.
- Konumlarınızı kategorize etmek için renkli gruplar oluşturabilir, düzenleyebilir ve silebilirsiniz.
- Bir konumu düzenlerken, oluşturduğunuz bu gruplardan birine atayabilirsiniz.

---

### 4. Rota Planlama ve Kullanımı
Uygulama, kayıtlı konumlarınızdan dinamik rotalar oluşturmanıza olanak tanır.

#### 4.1. Rota Oluşturma
- Harita ekranında üst menüdeki **yol tarifi (directions)** ikonuna tıklayın.
- **Gruptan Seç:** Belirli bir gruptaki tüm konumları içeren bir rota oluşturur.
- **Manuel Seçim:** Listeden istediğiniz konumları seçerek özel bir rota oluşturmanızı sağlar.
- Rota oluşturulduğunda, uygulama konumları mevcut konumunuza göre en verimli şekilde sıralar.

#### 4.2. Rota Kaydetme ve Yükleme
- **Kaydetme:** Rota özeti ekranında **kaydet (save)** ikonuna basarak mevcut rotayı isimlendirip kaydedebilirsiniz.
- **Yükleme:** Harita ekranında üst menüdeki **rota (route_sharp)** ikonuna basarak daha önce kaydettiğiniz rotaları yükleyebilirsiniz.

#### 4.3. Rota Özeti ve Kontrol Listesi
- Bir rota oluşturulduğunda veya aktif bir rota için üst menüdeki **özet (summarize)** ikonuna basıldığında bir özet ekranı açılır.
- Bu ekranda toplam mesafe, tahmini yol ve duraklama süreleri gibi bilgiler yer alır.
- **İhtiyaç Listesi:**
  - Rota üzerindeki konumlara eklediğiniz tüm ihtiyaçlar burada birleştirilmiş bir kontrol listesi olarak sunulur.
  - Bu liste **o anki seyahate özeldir**. İşaretlediğiniz maddeler (tikler), siz rotayı kapatana kadar korunur.
  - Rota kapatılıp yeniden açıldığında veya aynı rota şablonuyla yeni bir seyahat başlatıldığında, bu liste temizlenmiş (tiksiz) olarak gelir.
- **Özel Notlar:** Rota üzerindeki konumlara ait tüm özel notlar da bu ekranda bir arada gösterilir.

---

### 5. Aktif Rota Takibi
Bir rotayı başlattığınızda, harita ekranı seyahatinizi aktif olarak takip eder.

- **Gidilecek Yol:** Henüz gitmediğiniz yol parçaları haritada **açık gri** renkte gösterilir.
- **Gidilmiş Yol:** Bir durağa vardığınızda, o durağa kadar kat ettiğiniz yolun rengi kalıcı olarak **canlı maviye** döner.
- **Ulaşılan Duraklar:** Bir durağa 50 metreden daha fazla yaklaştığınızda, o durağın pini kalıcı olarak **yeşil** olur.
- **Bildirimler:** Bir durağa yaklaştığınızda, o konumla ilgili (varsa) Wikipedia özeti içeren bir bildirim alırsınız. Ayrıca bir durakta planladığınız süreden daha fazla kalırsanız süre hatırlatma bildirimi alırsınız.

#### 5.1. Ulaşılan Konumlar Geçmişi
- Üst menüdeki **geçmiş (history)** ikonu ile daha önce bir rota sırasında ulaştığınız ve bildirim aldığınız tüm konumların bir listesine ve Wikipedia linklerine erişebilirsiniz.
